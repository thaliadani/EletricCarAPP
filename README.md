# EletricCarApp

## Descrição

O EletricCarApp é um aplicativo Android que permite aos usuários visualizar uma lista de carros elétricos e calcular a autonomia de um veículo elétrico com base no preço por quilowatt-hora (kWh) e na distância percorrida.

## Funcionalidades

* **Lista de Carros Elétricos:** O aplicativo exibe uma lista de carros elétricos em abas.
* **Cálculo de Autonomia:** Os usuários podem inserir o preço por kWh e a distância percorrida para calcular o custo por quilômetro.
* **Interface Simples:** O aplicativo possui uma interface de usuário limpa e intuitiva, com abas para facilitar a navegação.

## Telas

* **Tela Principal:** Exibe as abas com a lista de carros elétricos e um botão de ação flutuante.
* **Tela de Cálculo de Autonomia:** Contém campos para inserir o preço do kWh e a distância a percorrer, e exibe o resultado do cálculo.

## Como usar

1.  Abra o aplicativo para ver a lista de carros elétricos.
2.  Toque no botão de ação flutuante para abrir a tela de cálculo de autonomia.
3.  Insira o preço por kWh e a distância que deseja percorrer.
4.  Toque no botão "Calcular" para ver o custo por quilômetro.
