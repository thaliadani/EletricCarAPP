# EletricCarApp

## Descrição

O EletricCarApp é um aplicativo Android que permite aos usuários visualizar uma lista de carros elétricos, calcular a autonomia de um veículo elétrico com base no preço por quilowatt-hora (kWh) e na distância percorrida, e favoritar carros.

## Funcionalidades

* **Lista de Carros Elétricos:** O aplicativo exibe uma lista de carros elétricos em abas.
* **Favoritar Carros:** Os usuários podem marcar seus carros preferidos como favoritos.
* **Aba de Favoritos:** Os carros favoritados são exibidos em uma aba separada para fácil acesso.
* **Cálculo de Autonomia:** Os usuários podem inserir o preço por kWh e a distância percorrida para calcular o custo por quilômetro.
* **Interface Simples:** O aplicativo possui uma interface de usuário limpa e intuitiva, com abas para facilitar a navegação.

## Telas

* **Tela Principal:** Exibe as abas com a lista de carros elétricos ("Carros" e "Favoritos") e um botão de ação flutuante.
* **Tela de Cálculo de Autonomia:** Contém campos para inserir o preço do kWh e a distância a percorrer, e exibe o resultado do cálculo.

## Como usar

1.  Abra o aplicativo para ver a lista de carros elétricos na aba "Carros".
2.  Toque no ícone de estrela para favoritar um carro.
3.  Acesse a aba "Favoritos" para ver a lista de carros que você favoritou.
4.  Toque no botão de ação flutuante para abrir a tela de cálculo de autonomia.
5.  Insira o preço por kWh e a distância que deseja percorrer.
6.  Toque no botão "Calcular" para ver o custo por quilômetro.
